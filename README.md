🤖 LLM Installer for Ubuntu WSL2A production-hardened, automated setup script to deploy a full Large Language Model (LLM) environment on Ubuntu WSL2. This installer streamlines the compilation of llama.cpp and provides a curated catalogue of the latest high-performance models.🚀 OverviewThis script is designed for developers and AI enthusiasts who want a "one-command" setup for local LLMs. It handles everything from system dependencies and CUDA toolkits to model quantization selection and hardware-specific performance grading.Core StackEngine: llama.cpp (Built from source with CUDA support).Models: Curated GGUFs including Qwen 3.5, Llama 3.3, Phi-4, and Gemma 4.Integration: Support for Hermes, Goose, OpenCode, and AutoAgent.✨ Key Features🔒 Security First: Safe extraction of HuggingFace and GitHub tokens from .bashrc without risky sed or grep injections.📟 Smart Hardware Detection: Automatically calculates RAM/VRAM and assigns a Hardware Grade (S–F) to models to prevent Out-of-Memory (OOM) errors.⚡ Automated Optimization: * Builds llama.cpp with NVIDIA CUDA acceleration (if a GPU is detected).Configures optimal Context Windows and Jinja templates per model.🔄 Model Switcher Mode: Run the script with SWITCH_MODEL_ONLY=1 for a lightweight model management experience without re-checking system dependencies.🌐 Flexible Downloads: Install from the curated catalogue, provide a direct HuggingFace URL, or browse any HF repository.📋 Hardware Grading LogicThe installer analyzes your system to determine how well a model will run:GradeMeaningExperienceSPerfect FitRuns at maximum speed with room to spare.AGreatSmooth performance, highly recommended.B/CTight FitFunctional, but expect slower tokens-per-second.FHeavyLikely to crash or resort to extremely slow CPU offloading.🛠️ Installation1. PrerequisitesWindows 10/11 with WSL2 (Ubuntu) installed.NVIDIA Drivers installed on the Windows host (for GPU acceleration).2. Run the InstallerOpen your WSL2 terminal and run:Bashgit clone https://github.com/mettbrot0815/llm-installer.git
cd llm-installer
chmod +x install.sh
./install.sh
3. Model Switcher (Fast Mode)If you have already installed the environment and just want to download or swap a model:BashSWITCH_MODEL_ONLY=1 ./install.sh
📖 Usage GuideToken ManagementUpon first run, the script will prompt for:HuggingFace Token: Required for gated models (like Llama) and higher download rate limits.GitHub Token: Used for authenticated repository cloning to avoid API rate limiting.Catalogue HighlightsThe installer provides a curated list of models including:Qwen 3.5 (0.8B to 30B): Excellent for coding and general reasoning.Carnice-9b: A specialized Hermes-tuned model for agentic tasks.Llama 3.3 70B: High-end performance for users with 24GB+ VRAM.DeepSeek R1 Distill: State-of-the-art reasoning capabilities.⚙️ Technical DetailsPython: Automatically installs Python 3.11 via the deadsnakes PPA if not present.CUDA: Installs CUDA Toolkit 12.6 specifically optimized for WSL2.Storage: Models are stored in ~/llm-models to keep your home directory organized.Clean PATH: Automatically strips Windows /mnt/* paths during execution to prevent build conflicts.🤝 ContributingContributions are welcome!
